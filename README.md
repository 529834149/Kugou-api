## 酷狗音乐API接口整理

>基于酷狗音乐整理的音乐API接口数据

### 音乐新歌榜

__说明:__ 获取新歌曲榜单

__必选参数:__
`page`: json

__接口地址:__ `http://m.kugou.com/?json=true`



###  音乐排行榜

__说明:__ 获取音乐排行榜

__必选参数:__

__接口地址:__ `http://m.kugou.com/rank/list&json=true`

__返回数据__


###  排行版分类歌曲列表

__说明:__ 获取音乐排行榜

__必选参数:__
`rankid` 排行榜分类下id
`json` 返回类型


__接口地址:__ `http://m.kugou.com/rank/info/?rankid=8888&page=1&json=true`

__返回数据__
